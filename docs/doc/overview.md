# SecScore

SecScore 是一款教育积分管理软件，基于 Electron + React + TypeScript 开发，用于管理学生名单、记录加/扣分、查看排行榜与结算历史，并提供权限保护与数据备份。

## 主要功能

- 学生管理：添加/删除学生；通过 xlsx 批量导入名单（支持导入前预览与选择姓名列）
- 积分管理：加分/扣分；快捷理由一键填充理由与分值；支持撤销最近记录并回滚积分
- 理由管理：维护预设理由（分类、理由内容、预设分值）
- 排行榜：按今天/本周/本月查看积分变化；支持导出排行榜为 XLSX；可查看单个学生的操作记录
- 结算与历史：结算并重新开始（归档阶段 + 清零积分）；结算历史查看每个阶段排行榜
- 系统设置：主题切换；日志查看/导出/清空；数据导入/导出（JSON）；密码保护（管理密码/积分密码）与找回字符串

## 权限模型

- 管理权限：学生管理、理由管理、结算、数据管理等全部功能
- 积分权限：仅允许积分相关操作
- 只读：仅查看，不允许修改
