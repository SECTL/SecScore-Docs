## 安装与启动

### Windows

- 安装版：按安装向导完成安装后，从开始菜单启动 SecScore
- 便携版：解压后直接运行（建议把程序放在固定目录，便于更新与备份）

## 第一次使用（推荐顺序）

1. 确认右上角当前权限：管理权限 / 积分权限 / 只读
2. 如设置了密码，在右上角点击“输入密码”解锁
3. 学生管理 → 导入名单（.xlsx）
4. 积分管理 → 选择学生 → 加分/扣分 → 填写理由 → 确认提交
5. 排行榜 → 切换统计范围（今天/本周/本月）→ 需要时导出 XLSX
6. 系统设置 → 数据管理 → 导出 JSON（建议定期备份）
