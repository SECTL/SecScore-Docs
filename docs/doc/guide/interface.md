# 界面介绍

## 界面概览

SecScore 是一款桌面端班级积分管理软件，基于 Electron + React + TypeScript 开发。整体交互以“列表/表格 + 操作面板”为主，支持快捷加扣分、原因预设、排行榜与结算归档等常用流程。

## 主题与显示

- 支持浅色/深色主题切换
- 常用展示区域包括：学生列表、积分记录列表、排行榜表格、结算历史

## 主要功能入口

- 学生管理：维护学生名单，支持从 xlsx 导入（导入前预览与选择姓名列）
- 积分管理：选择学生后加分/扣分，可使用预设理由一键填充分值与说明
- 排行榜：按今天/本周/本月统计积分变化，支持导出为 XLSX
- 结算与历史：结算并进入新阶段，历史阶段可回看排行榜与记录
- 系统设置：主题、日志、数据导入/导出（JSON）、密码与权限相关设置

## 权限提示

界面会显示当前权限状态（管理 / 积分 / 只读）。当功能被权限限制时，入口仍可见但操作会被拦截并提示需要解锁或更高权限。
