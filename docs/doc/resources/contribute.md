# 开发与贡献

SecScore 欢迎任何形式的贡献，包括但不限于：反馈问题、改进文档、提交代码、完善翻译等。

## 反馈问题

- 提交入口：https://github.com/SECTL/SecScore/issues
- 建议附上：系统版本、SecScore 版本、复现步骤、截图/日志、期望行为与实际行为

## 提交代码

1. Fork 仓库：https://github.com/SECTL/SecScore
2. 创建分支并开发
3. 提交 PR，并在描述中说明变更目的与影响范围

## 改进文档

本仓库为 SecScore 文档站。你可以直接在对应页面中进行修订并提交 PR。
