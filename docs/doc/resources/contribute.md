# 开发与贡献

SecScore 欢迎任何形式的贡献，包括但不限于：
- 反馈问题
- 提交代码
- 改进文档
- 完善翻译

## 反馈问题

- 提交入口：[Github Issues](https://github.com/SECTL/SecScore/issues)
- 我们希望您尽可能提供以下信息：
  - 操作系统版本
  - SecScore 版本
  - 问题描述
  - 复现步骤（如果您有办法复现）
  - 截图/日志
  - 期望行为与实际行为
    ::: tip 提示
    您提供的信息越多，越有利于问题的解决。
    :::

## 提交代码

1. Fork 仓库：[SECTL/SecScore](https://github.com/SECTL/SecScore)
2. 创建分支并开发
3. 提交 PR，并在描述中说明变更目的与影响范围

## 改进文档

本仓库为 SecScore 文档站。你可以直接在对应页面中进行修订并提交 PR。
