> 🔗 **外部联动与集成**
> 
> SecScore 当前主要聚焦于“学生名单 + 积分记录 + 排行榜 + 结算归档”的本地工作流。外部课表联动、URL 协议等集成功能会随版本迭代逐步补齐。

## 目前支持范围

- 站内联动：在不同页面间快速跳转、从学生列表快速进入加/扣分
- 数据导入/导出：通过 JSON 导入/导出进行跨设备迁移与备份

## 规划中的方向（不保证）

- 课表联动：根据上课时间自动切换班级/阶段
- 外部通知：与教学工具或桌面通知系统联动

如需集成能力或有具体使用场景，可通过 [GitHub Issues](https://github.com/SECTL/SecScore/issues) 提交需求。
