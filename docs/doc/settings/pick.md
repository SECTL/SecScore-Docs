> 🧮 **记录每一次加/扣分**
>
> 通过“选择学生 → 选择原因/填写说明 → 输入分值 → 确认提交”的流程，把日常课堂表现稳定落到可追溯的积分记录里。

## 🎯 记分流程

1. 在顶部选择当前班级（如你维护了多个名单）
2. 在学生列表中选中目标学生
3. 选择“加分”或“扣分”
4. 选择一个理由预设（可选）或手动填写理由
5. 输入分值并确认提交

## ➕➖ 加分与扣分

- **加分**：用于奖励加分、表扬、课堂回答等场景
- **扣分**：用于纪律、作业、迟到等扣分场景
- **分值建议**：保持分值颗粒度一致（例如以 1 / 2 / 5 为主），方便后续统计与沟通

## 🏷️ 理由预设（快捷填充）

- 选择预设后会自动填充理由与默认分值，减少重复输入
- 建议按“作业/纪律/课堂表现/奖励”等分类维护，便于快速查找

## 🧾 记录列表与撤销

- **查看记录**：按时间顺序查看加/扣分明细（学生、分值、理由、时间）
- **撤销最近记录**：误操作时可撤销最近一条记录，并自动回滚对应积分

## 💡 使用建议

- 先维护好“常用理由预设”，再开始大规模记分
- 每个阶段结束前先导出一次 JSON 备份，再进行结算
- 需要查看“阶段排名”和导出表格时，请前往 [排行榜](./notification.md)
