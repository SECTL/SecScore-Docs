> **Quick Start** - Download and run SecScore

## Download

- Download: https://stk.sectl.top/SecScore
- GitHub Releases: https://github.com/SECTL/SecScore/releases

## Install & Launch (Windows)

- Installer: run the setup and launch SecScore from Start Menu
- Portable: unzip to a fixed folder and run the executable

## First-time Checklist (Recommended)

1. Confirm current permission state: Admin / Score / Read-only
2. If passwords are enabled, click “Enter Password” to unlock
3. List Management → import roster from xlsx
4. Scoring → select student → add/subtract points → choose a reason → submit
5. Leaderboard → switch range (today / week / month) → export XLSX if needed
6. Settings → Data → export JSON regularly for backup
