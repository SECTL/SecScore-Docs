# ❓ 程序无法启动          

## **症状1**：修改设置后程序无法启动

**解决方案**：
- 1. 关闭程序（检查任务管理器确保完全退出）
- 2. 备份后删除应用数据/配置目录（以设置页中展示的路径为准）
- 3. 重新启动程序，确认能正常进入
- 4. 如问题复现，导出日志/诊断信息并通过 GitHub Issues 反馈

## **症状2**：启动后闪退或无窗口弹出                                              
**解决方案**：
- 1. 尝试重启电脑后再次启动
- 2. 如出现 DLL 缺失/运行库报错，尝试安装 [VC++ 运行库](https://aka.ms/vc14/vc_redist.x64.exe)
- 3. 仍无效时，建议重新安装最新版本（优先使用安装包）
- 4. 如仍存在问题，请附上日志与版本信息反馈给开发者
