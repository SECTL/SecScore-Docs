# 社区文章精选

这里汇集了社区成员分享的 SecScore 使用心得、技巧教程和创意应用。每一篇文章都代表着社区的智慧和实践经验。

## 如何提交你的文章

如果你有想分享的内容，欢迎通过以下方式提交：

1. 访问 [SecScore GitHub Issues](https://github.com/SECTL/SecScore-Docs/issues)
2. 创建一个新的 Issue，标题格式：`[文章推荐] 文章标题`
3. 在正文中提供：
   - 文章标题和简要介绍
   - 文章链接（如果是外部链接）
   - 或直接粘贴文章内容（如果愿意贡献到文档站）

## 我们欢迎的内容

- 使用技巧与快捷操作分享
- 班级管理场景的最佳实践
- 创意用法和实际案例
- 与其他工具的集成方案
- 教师工作流优化经验

期待你的分享！
