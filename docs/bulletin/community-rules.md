# 社区规范

> 本规则适用于 SECTL 组织下所有项目及相关社区。

欢迎来到 SECTL 组织社区！为了保证良好的交流环境，请在交流时遵守以下规则。

## 🚫 禁止行为

在社区交流时，请遵守相关法律法规，**严禁**发布以下内容：

1.  **违法违规**：涉黄、涉赌、涉毒、恐怖暴力、涉政等敏感或违法言论。
2.  **恶意内容**：病毒、木马、外挂等恶意软件或链接。
3.  **广告推销**：未经许可的商业广告或推广信息。
4.  **干扰秩序**：刷屏、恶意复读、频繁骚扰他人等干扰正常交流的行为。
5.  **不当言论**：人身攻击、辱骂、引战等破坏社区氛围的言论。

## 💡 提问与求助

为了更高效地解决您的问题，提问前请遵循以下建议：

1.  **先尝试自查**：查阅[官方文档](/doc/guide/start.html)、常见问题或使用搜索引擎。
2.  **提供详细信息**：
    *   **环境信息**：操作系统版本、软件版本。
    *   **问题描述**：发生了什么、如何复现（步骤）。
    *   **诊断信息**：提供错误截图、录屏或程序日志。
3.  **端正态度**：提问时请保持礼貌，遵循[《提问的智慧》](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way/blob/main/README-zh_CN.md)。

> ❌ **错误示范**：“软件打不开怎么办？”（无任何信息）
>
> ✅ **正确示范**：“我在 Windows 11 上使用 SecScore v1.0.0，点击启动后无反应，这是我的错误日志截图...”

## ⚖️ 违规处理

违反上述规定者，管理员将视情节轻重采取以下措施：
*   口头警告 / 撤回违规内容
*   禁言处理（24小时 - 30天）
*   永久移除出群 / 拉黑

希望大家共同维护一个友好、互助的社区环境！
