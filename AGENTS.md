## OVERVIEW

## CONVENTIONS

### Content Organization
- **Collections pattern**: Uses `collections.ts` for sidebar navigation
- **Bilingual structure**: Mirror directories (docs/ ↔ docs/en/)
- **Content types**: `doc` (guides), `post` (FAQ, bulletins)
- **File naming**: kebab-case.md (e.g., `basic-settings.md`)

### Markdown Frontmatter
```yaml
---
title: Page Title                    # Required
createTime: 2025/11/25 14:24:16     # Auto-generated by plume
---
```

### VuePress Configuration
- **Theme**: vuepress-theme-plume (v1.0.0-rc.187)
- **Bundler**: Vite
- **Icons**: Iconify provider (`lucide:*`, `material-symbols:*`, etc.)
- **Search**: Local search enabled
- **Features**: Icon syntax enabled, table alignment center

### Icon Syntax
```markdown
Icon format: icon-provider:icon-name
Examples: lucide:download, material-symbols:overview-outline-rounded
```

### Localization
- Chinese: `/` (default)
- English: `/en/`
- Both use same file naming, different content
